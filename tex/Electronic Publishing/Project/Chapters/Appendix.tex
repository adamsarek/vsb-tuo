\section{Vkládání grafů z GNUplot}
Graf matematické funkce $x^2$:

\begin{gnuplot}[terminal=pdf,terminaloptions=color]
    unset key
    set samples 1000
    set format '%g'
    set xlabel "x"
    set ylabel "y"
    set xrange [-10:10]
    set yrange [0:100]
    plot x**2
\end{gnuplot}
